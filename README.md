## 入门部署和体验 ##

```js
1. `git clone ` 仓库
2. `npm install` 安装依赖：`安装 grunt` 安装 `http-server`
3. 执行 `npm run build` 打包编译
4. 执行 `npm run dev` 开启本地预览
5. 重复 `npm run build` 页面刷新查看效果
```
